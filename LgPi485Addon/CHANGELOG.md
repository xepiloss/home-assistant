# Changelog

## 0.1.0 - 2026-04-25

첫 배포입니다.

- EW11 TCP 브리지를 통한 LG PI485 상태 수신 및 제어를 추가했습니다.
- Home Assistant MQTT discovery 기반 climate, switch, binary sensor, diagnostic sensor를 추가했습니다.
- 실내기별 전원, 운전 모드, 목표 온도, 팬 모드, 스윙 모드 제어를 지원합니다.
- 냉방 전용 기본 구성을 적용했습니다.
- 제어 명령 처리 중에는 상태 폴링을 잠시 멈추고 제어 명령을 우선 전송하도록 큐를 구성했습니다.
- 상태 폴링 기본값은 2초 주기, 실내기별 150ms 간격으로 설정했습니다.
- 상태 폴링 요청/응답 로그는 기본 비활성으로 두고, 실제 제어 요청/응답 로그만 기본으로 남기도록 했습니다.
- 배관 온도, raw frame, 에러 코드, 부하 계열 값을 진단 센서로 노출했습니다.
- 알 수 없는 패킷 캡처 기능을 추가했습니다.
- PI485/LGAP 경로에서 확인되지 않은 0.5도 목표온도와 습도는 지원하지 않습니다.
