# Changelog

## 1.1.1-dev - 2026-04-25

패킷 수집과 생활정보 분석을 계속하기 위한 개발용 애드온 배포입니다.

### Added

- 개발용 애드온 `Commax Wallpad Dev`를 추가했습니다.
- Dev 애드온 slug를 `commax_wallpad_dev`로 분리했습니다.
- 알 수 없는 패킷 수집 옵션을 추가하고 Dev 애드온에서는 기본 ON으로 설정했습니다.
- 알 수 없는 프레임과 프레이밍 중 버려진 바이트를 `/share/commax_unknown_packets.jsonl`에 JSON Lines 형식으로 저장합니다.
- 동일한 알 수 없는 패킷은 하나의 레코드로 묶고 `seen_at` 수신 시각 배열만 누적하도록 했습니다.
- 생활정보 `8F` 프레임도 값 변동 분석을 위해 `life_info_frame`으로 수집 파일에 저장합니다.
- 알려진 조회/상태성 패킷은 수집 대상에서 제외해 분석 대상 패킷만 남기도록 필터링했습니다.
- `7F` 월패드 시간 패킷을 파싱하고 Home Assistant timestamp 센서로 등록합니다.
- 월간 검침 사용량의 월 판단에 월패드 시간을 우선 사용하도록 개선했습니다.
- `8F` 생활정보 응답은 아직 의미를 확정하지 않고 진단용 raw 패킷 센서와 비교용 attributes로 발행합니다.

### Notes

- 배포용 `Commax Wallpad`와 Dev 애드온을 같은 EW11에 동시에 연결하지 마세요.
- `8F` 생활정보 패킷의 날씨/기온/미세먼지 매핑은 실제 월패드 표시값과 더 비교한 뒤 확정할 예정입니다.
- 같은 패킷은 중복 본문을 저장하지 않지만 수신 시각은 계속 누적되므로 장기 수집 후에는 필요 시 백업하거나 정리해 주세요.

## 1.1.0 - 2026-04-25

이번 릴리스는 실사용 안정성과 Home Assistant 사용감을 크게 다듬은 기능 업데이트입니다.

### Added

- EW11 TCP 스트림 프레이밍을 추가해 쪼개지거나 붙어서 들어오는 패킷을 안정적으로 처리합니다.
- 원격 검침 `F7` 가변 길이 프레임을 지원합니다.
- 월간 검침 센서를 추가했습니다.
  - 이번달 전기 사용량
  - 이번달 수도 사용량
  - 이번달 온수 사용량
  - 이번달 난방 사용량
  - 이번달 가스 사용량
- 월 중간 설치 사용자를 위한 월간 사용량 보정 옵션을 추가했습니다.
- 명령 송신/수신 로그에 디바이스 종류, ID, 명령 종류, 응답 시간을 표시합니다.

### Changed

- 같은 디바이스에 빠르게 연속 명령이 들어올 때 최신 명령 중심으로 처리되도록 개선했습니다.
- 조명 밝기 명령 직후 Home Assistant가 보내는 중복 `ON` 명령을 억제합니다.
- 빠른 조작 시 ACK/STATE 매칭 정확도를 높여 불필요한 재전송 가능성을 줄였습니다.
- 명령 재전송 시점을 실제 송신 기준으로 조정해 빠른 연속 제어 안정성을 개선했습니다.
- 종료 처리에서 상태 저장과 availability publish가 더 안정적으로 수행되도록 정리했습니다.
- 검침, 공기질, 주차 센서에 값 성격에 맞는 아이콘을 지정했습니다.

### Fixed

- TCP 조각 수신으로 검침 프레임이 유실되거나 unknown packet으로 분리되던 문제를 수정했습니다.
- 짧은 공기질 패킷이 들어와도 파서가 죽지 않도록 방어 코드를 추가했습니다.
- 검침 장비가 분리된 환경에서도 원격 검침 프레임이 정상 파싱되도록 개선했습니다.

### Notes

- 월간 검침 센서는 애드온이 전체 누적값을 기준으로 내부 월초 기준값을 저장해 계산합니다.
- 월 중간에 설치했다면 애드온 옵션의 `monthly_*_usage` 값에 현재까지의 이번 달 사용량을 입력하면 됩니다.
- 기존 사용자도 다음 검침/공기질/주차 패킷 수신 시 discovery가 갱신되어 새 아이콘을 받을 수 있습니다.
